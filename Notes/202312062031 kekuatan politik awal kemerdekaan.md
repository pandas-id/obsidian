# Note

Menurut Harberf Feith ada lima kekuatan politik besar pada awal kemerdekaan indonesia,yaitu:
1. Nasionalisme (PNI)
2. Islamisme (NU dan Masyumi)
3. Komunisme (PKI)
4. Sosialisme (PSI)
5. Tradisional Jawa (Partai Indonesia Raya)
